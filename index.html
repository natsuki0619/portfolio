── SKILL ──

HTML/CSS

HTMLとCSSを使って基本的なウェブページの構築ができます。簡単なレイアウトやスタイルの設定が可能です。 チュートリアルやオンラインコースを通じてHTMLとCSSの基礎を学習し、実践的なプロジェクトでスキルを磨いています。 最新のトレンドやウェブデザインの基本原則に基づいて、見やすく使いやすいページを作成することに興味を持ち、積極的に学習しています。

Java

Java言語を使用してバックエンド開発を行うことができます。基本的なプログラミング概念（制御構造、データ型、メソッド、クラスなど）に精通しています。 Spring Frameworkを利用したWebアプリケーションの開発経験があります。Spring Bootを使用してRESTful APIを作成し、データベースとの連携を実装しました。 データベース操作に関しては、JPA/Hibernateを使用してオブジェクト関係マッピング（ORM）を行うことができます。MySQLのリレーショナルデータベースと統合する経験があります。 プログラミングに関する情報を常に学習し、技術的な課題に対処するための問題解決能力を向上させることに興味があります。

Javascript

JavaScriptの基本的な文法や構文に理解があります。変数、データ型、演算子、制御構造（if文やforループ）、関数などの基本を学習しました。 イベント処理について基本的な知識があります。ユーザーの操作に応じて動作するようなイベントリスナーを追加したり、イベントハンドラーを実装したりすることができます。 モダンなJavaScriptやフレームワークにも興味があり、将来的にはReactやVueなどの学習を目指しています。

Spring Boot

Spring Bootのアノテーションや設定について基本的な知識があります。@Controller、@Service、@Repository、@Autowiredなどの基本的なアノテーションを使用して、Spring Bootアプリケーションを構築できます。Webアプリケーションの開発RESTful APIの作成や操作、データベースとの連携、セキュリティの実装など、SpringBootを使用したWebアプリケーションの基本的な開発ができます。CRUD操作やデータの検索、更新、削除などの基本的な操作を実装できます。

Word Press

Word Pressを使用して、ブログや簡単なウェブサイトを作成できます。操作インターフェースと豊富なテーマ・プラグインの選択肢を活用し、ウェブサイトを構築できます。

VBA

VBAを独学で学び、業務に活用してきました。業務効率化やデータ処理の自動化にVBAを活用し、数多くのタスクを効率化しました。具体的には、Excelを用いたデータ処理やレポート作成の自動化、Accessを用いたデータベース管理システムの構築などを行いました。VBAを使用して複雑な業務プロセスを自動化することで、業務の効率化と生産性向上に貢献しました。

Python

基本的な構文やデータ構造を習得し、Djangoフレームワークを使ってウェブアプリケーションの開発に取り組みました。チャットボットの開発においては、ユーザーとの対話インターフェースを構築し、自然な会話を実現するための技術や手法を学びました。この経験を通じて、PythonとDjangoを活用して実用的なプロジェクトを構築する能力を身につけました。

── ABOUT ME──

松原 なつき

沖縄県出身、愛媛県在住。医療業界で10年間のキャリアを積んできました。
電子カルテの導入に携わる中で、業務の効率化を図るためVBAを利用する機会があり、 それが私のプログラミングへの関心を高めるきっかけとなりました。
その後、技術職に転身しましたが、ソフトウェアを使用する技術職であったため、プログラミングに対する興味はますます強まっていきました。
そこで、オンラインプログラミングスクールに入学し、 WebエンジニアJavaコースを2024年5月に卒業。コースの目標は1から1人でWebアプリケーションを作成することであり、 週に40時間以上の学習時間を確保し、着実にスキルを磨いてきました。
趣味はお菓子作りや、一緒に住んでいるインコとのひとときを楽しむことです。
